# AI Fake Interview Detection System
# 📌 Project Overview

The AI Fake Interview Detection System is a full-stack web application designed to monitor and analyze online interview sessions using Artificial Intelligence, Computer Vision, and Behavioral Analysis techniques.

The system aims to identify suspicious activities and detect potentially fake interview behavior in real time. It combines webcam-based face analysis, lip-sync monitoring, behavioral tracking, and session management into a unified interview monitoring platform.

This project uses a React frontend for the user interface and a Django backend for AI processing and API management.

# 🚀 Features

# 🎥 Real-Time Interview Monitoring
Live webcam-based interview monitoring
Continuous face tracking during interviews
Session activity observation

# 🧠 AI-Based Suspicious Behavior Detection
Head movement analysis
Eye movement tracking
Multiple face detection support
Suspicious activity scoring

# 🎤 Lip Sync Analysis
Detects mismatch between audio and lip movement
Helps identify possible fake or pre-recorded interview attempts

# 📊 Dashboard Analytics
Interview session summaries
Suspicious activity reports
Question performance tracking
Real-time status indicators

# 👤 Authentication System
User login and registration
Session management
Protected dashboard access

# 📁 Interview History
Previous session records
Session detail pages
Activity logs and reports
